まとめて：
一般的な賃金体系は、1時間ごとに1072円ずつ増えるため、線形的に増加する。
一方、再帰的な賃金体系は、前の時間の時給をもとに「前の時給 * 2 - 50」で次の時給が決まるため、指数関数的な増加に近い。
そのため、最初は再帰的な賃金体系のほうが少ないが、時間が長くなると増加量が大きくなり、8時間で一般的な賃金体系を上回る。

計算結果は以下のようになる。

| 時間 | 一般的な賃金体系 | 再帰的な賃金体系 |
------------------------------------------
　| 1時間 | 1072円 | 100円 |
　| 2時間 | 2144円 | 250円 |
　| 3時間 | 3216円 | 500円 |
　| 4時間 | 4288円 | 950円 |
　| 5時間 | 5360円 | 1800円 |
　| 6時間 | 6432円 | 3450円 |
　| 7時間 | 7504円 | 6700円 |
　| 8時間 | 8576円 | 13150円 |

7時間までは一般的な賃金体系のほうが高いが、8時間働くと再帰的な賃金体系のほうが高くなる。
再帰的な賃金体系のほうが稼げるようになるのは8時間である。
